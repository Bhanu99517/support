# 🛠️ Jec-Forge Support

> The official support and community onboarding repository for the **Jec-Forge GitHub Organization**.

---

## 📌 What is this repository?

`Jec-Forge/support` is the central hub for:

- Requesting an **invitation** to join the Jec-Forge GitHub Organization
- Reporting **issues** related to the community or organization
- Asking **questions** about projects, contributions, or membership
- Providing **feedback** to help improve the organization

---

## 🚀 How to Join Jec-Forge

Want to become a member of the Jec-Forge community? Follow these steps:

1. **Open a new Issue** in this repository
2. Use the **"Invite me to the community"** issue template
3. Fill in all required fields (Name, Discord Username, Background)
4. Select the relevant checkboxes that describe your skills
5. **Wait for a maintainer** to review your request and send an invite

> ✅ Once approved, you will receive a GitHub organization invite via email and notifications.

| Template | Purpose |

## 👥 Who Can Join?

Jec-Forge welcomes developers from all backgrounds. Members typically include:

- 🎓 Students and graduates of JEC (Jyothy Engineering College)
- 🌐 Web Developers
- 📱 Flutter / Mobile Developers
- ⚙️ DevOps Engineers
- 🔒 Cyber Security Enthusiasts
- 🤖 AI / ML Engineers
- And anyone passionate about open-source!

---

## 🤝 Contributing

Once you're a member, you can contribute to any Jec-Forge project by:

1. Forking the repository you want to contribute to
2. Creating a new branch (`git checkout -b feature/your-feature`)
3. Making your changes and committing them
4. Opening a **Pull Request** with a clear description

Please follow the individual project's `CONTRIBUTING.md` guidelines if available.

---

## 📜 Code of Conduct

All members are expected to follow our community standards:

- Provide constructive feedback
- Help others and collaborate openly
- Zero tolerance for harassment or discrimination
